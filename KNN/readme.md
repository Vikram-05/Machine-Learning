# Titanic Survival Prediction with KNN

This folder contains a machine learning project that predicts passenger survival on the Titanic using the k-Nearest Neighbors (KNN) algorithm.

## Contents

- **KNN.ipynb** – The primary Jupyter notebook documenting the data loading, cleaning, feature engineering, model training, and evaluation steps.
- **Dataset** – The Titanic dataset used for training and testing the KNN classifier.

## Usage

1. Open the `Untitled2 (1).ipynb` notebook in Jupyter Lab or VS Code.
2. Ensure required packages (`pandas`, `scikit-learn`, etc.) are installed in your Python environment.
3. Run each cell sequentially to reproduce the analysis and model results.

## Notes

- Data preprocessing includes handling missing values, encoding categorical features, and scaling numerical data.
- Hyperparameter tuning may improve the KNN model's performance; try varying the number of neighbors or distance metrics.
- Use the notebook as a starting point for experimenting with other classification algorithms.
